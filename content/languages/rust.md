## Zmq - bindings for the libzmq

| Github   | https://github.com/erickt/rust-zmq                      |
|----------|---------------------------------------------------------|
| Crate    | https://docs.rs/crate/zmq                               |
| Examples | https://github.com/erickt/rust-zmq/tree/master/examples |

### Installation

#### Via cargo add:
```bash
cargo add zmq
```

#### Via Cargo.toml file:
```toml
[dependencies]
zmq = "0.10.5"
```
